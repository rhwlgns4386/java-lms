# 학습 관리 시스템(Learning Management System)
## 기능 요구사항
- 질문 삭제 처리
  - 질문을 삭제하는 것이 아니라 질문의 삭제 상태를 true 로 변경하여 처리한다.
  - 질문의 답변도 삭제 상태를 true 로 변경하여 함께 처리한다.
  - 질문, 답변 삭제 이력 정보를 이력에 남긴다.
- 질문 삭제 가능 조건
  - 로그인 사용자와 질문한 사람이 같은 경우
  - 답변이 없는 경우
  - 질문자와 답변글의 모든 답변자가 같은 경우

## 리팩터링 요구사항
